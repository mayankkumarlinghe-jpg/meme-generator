# AI Meme Generator Pro

AI Meme Generator Pro is an interactive web application that allows users to quickly and easily create memes using popular templates or their own images. It features an integrated AI assistant powered by Google Gemini to help generate funny and context-aware meme captions!

## Features

- **Meme Templates**: Browse and select from popular meme templates via the Imgflip API.
- **Image Upload**: Upload your own images (JPG, PNG, GIF) up to 5MB to use as meme backgrounds.
- **Meme Editor**: Customize your meme with top and bottom text, font size, font family, text color, and stroke settings.
- **AI-Powered Captions**: Leverage Google Gemini AI to automatically generate funny captions, suggest themes, or improve your current text.
- **Export & Share**: Download your generated meme directly to your device or share it.

## Technologies Used

- Vanilla HTML, CSS, and JavaScript
- [Imgflip API](https://imgflip.com/api) to fetch popular meme templates
- Google Gemini AI for AI-generated meme captions and suggestions
- FontAwesome for UI icons

## Getting Started

1. Open `index.html` in your favorite web browser.
2. Select a meme template from the grid or upload your own image.
3. Use the Editor Section to add your text manually or use the **AI Assistant** buttons to auto-generate hilarious captions.
4. Customize the font style, colors, and layout as needed.
5. Click **Download Meme** to save your masterpiece!
